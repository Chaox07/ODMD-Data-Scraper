# ODMD-Data-Scraper
This project was created to address a measurement issue in Turkey’s GFCF decomposition, specifically within the machinery and equipment component.

A significant share of this category is driven by light commercial vehicle purchases, which can distort the interpretation of real investment dynamics.

To correct for this, the scraped series of light commercial vehicle sales (x) is used in a regression framework with the machinery and equipment component (y). The residual component is then interpreted as a proxy for underlying machinery and equipment investment, net of vehicle purchases.
